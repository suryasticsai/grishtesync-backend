## CORRECT GRADIO PATTERNS

### Interface (simple apps):
```python
demo = gr.Interface(
    fn=my_function,
    inputs=[gr.File(type="file_path"), gr.Textbox()],
    outputs=gr.Textbox(),
    title="App Title"
)
if __name__ == "__main__":
    demo.launch(server_name="0.0.0.0", server_port=7860)
```

### Blocks (multi-step apps):
```python
with gr.Blocks() as demo:
    gr.Markdown("# Title")
    # components...
if __name__ == "__main__":
    demo.launch(server_name="0.0.0.0", server_port=7860)
```

### Timer (replaces every= parameter):
```python
with gr.Blocks() as demo:
    t = gr.Timer(5, active=True)
    t.tick(my_function, inputs, outputs)
```

### File component (Gradio 5+):
```python
gr.File(type="file_path", file_types=[".pdf", ".txt"])
```

### ChatInterface (Gradio 5+):
```python
demo = gr.ChatInterface(
    fn=chat_function,
    title="Chat App",
    # submit_btn and stop_btn accept bool or string, not gr.Button instances
)
```

### DataFrame (Gradio 5+):
```python
gr.DataFrame(max_height=400)  # NOT height
```

## requirements.txt RULES

Correct examples:
```

transformers>=4.24.0

```

Wrong examples:
```

gradio4>=4.0.0        # version in name
```
